----------------------------
Team-09 Egalloc Application
----------------------------

To run the application:
1) Open a terminal by clicking on the terminal icon or searching for the terminal
2) The terminal automatically opens to the following directory:  /home/student
3) A bash script is already in the directory so run the script using the following command:
	./run.sh
4) Tomcat should run and open Google Chrome
5) Enter the following address in the web browser:
	localhost:8080/Egalloc/firstpage.jsp
6) Then the Egalloc application should start.
